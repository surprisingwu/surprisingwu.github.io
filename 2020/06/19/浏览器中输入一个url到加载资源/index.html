浏览器中输入一个url到加载资源

浏览器中输入一个url, 回车后中间经历那些流程?

大致的流程主要有一下几步:

  1. DNS 查询
  2. TCP 三次握手,建立连接
  3. HTTP 发送数据
  4. 服务端处理和响应数据
  5. 客户端接收数据
  6. TCP关闭连接.四次挥手
  7. 浏览器渲染
